# Personal OS Capture — Chrome Extension

One-click capture of any webpage to your Personal OS (reading list, second brain, tasks, or chat).

## Install (takes 30 seconds)

1. Open Chrome and go to `chrome://extensions`
2. Toggle **Developer mode** on (top-right corner)
3. Click **Load unpacked**
4. Select this `extension/` folder
5. The 🌸 icon appears in your toolbar — pin it for easy access

## How it works

Click the toolbar icon on any page → a preview shows the page title and URL → click **Capture this page** → a Personal OS popup opens → choose where to send it:

- **Reading List** — saves as "want to read"
- **Second Brain** — drops into your inbox
- **New Task** — creates an active task with the page title
- **Send to Chat** — opens Claude with the URL pre-filled

## Notes

- You must be logged into Personal OS in Chrome for the popup to work
- No separate auth needed — uses your existing browser session
- Updates to the extension folder are picked up automatically after clicking the refresh icon on `chrome://extensions`
